# QuerySmith – AI Powered SQL Query Generator

QuerySmith is an AI-powered tool that converts natural language questions into SQL queries and visualizes database insights.

## Features
- Natural Language → SQL query generation
- AI-powered query understanding
- Interactive data visualization
- Query insights and explanations
- Clean full-stack architecture

## Tech Stack
Backend:
- Flask
- Python

Frontend:
- React
- Tailwind CSS

AI Layer:
- LLM based query interpretation
## Architecture

User Question  
↓  
React Frontend  
↓  
Flask Backend API  
↓  
AI Query Generator  
↓  
SQL Database  
↓  
Visualization & Insights


### HOW TO RUN
 
Backend
cd backend
pip install -r requirements.txt
python app.py

Frontend
cd frontend
npm install
npm start
